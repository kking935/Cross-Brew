## Development
To start for development:
```
# with npm
npm run dev

# or with yarn
yarn dev
```

During development, write your typescript code in the `app.ts` file.

## Production
To build for production:
```
# with npm
npm run build

# or with yarn
yarn build
```

This will compile the typescript code into javascript and add it to the `dist` folder
